Getting Started

The first step to using Git Achievements is configuring your organization(s)/repository/repositories to use Git Achievements. Github has a nice page about adding a webhook here. The steps are simple.

  1. Follow he Github steps for setting up a webhook.
  2. For the endpoint, plug in http://git-achievements.com/services/hooks/web.
  3. Login to the site to start creating and tracking achievements!


Read the Getting Started section of the site to figure out how to configure Git Achievements for your organization or start achieving in your projects/git experience. If you have any issues or more detailed questions, visit the Github project page.
